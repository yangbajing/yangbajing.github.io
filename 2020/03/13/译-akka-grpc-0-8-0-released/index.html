译：Akka gRPC 0.8.0 Released

2020-03-13

亲爱的 hakker 们！

我们高兴的公布 Akka gRPC 0.8.0 版本！gRPC 是请求/响应和流式处理（非持久化）场景的传输机制。参见 Why gRPC？ （获得）何时使用 gRPC 作为传输机制的更多信息。这个版本引入了许多令人兴奋的新特性，并使我们更接近 1.0.0 （版本），我们预计在数据周内发布。

主要的改进包括：

  • 基本支持 gRPC Server Reflection #380，允许交互式/动态客户端发现（gRPC）服务和方法，基于 Cloudstate 的实现。
  • 支持 gRPC-Web #695，生成 gRPC 可用于浏览器 JavaScript 客户端，由 @timw 贡献。
  • 支持自定义尾随响应头 #838，例如：可用于实现 rich error response ，由 @drmontgomery 贡献。
  • 改善了在多个后端实例上通过 Akka Discovery #81 的客户端负载均衡发现（机制）。

有关所有变更的描述，请见 release overview。
